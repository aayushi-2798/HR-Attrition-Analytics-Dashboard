# HR Attrition Analytics Dashboard



##### \## Project Overview



This project analyzes employee attrition patterns to identify the key factors influencing employee turnover.



The analysis was performed using SQL, Python and Excel Dashboarding techniques.



\---



###### \## *Tools Used*



\- SQL

\- Python (Pandas, Matplotlib)

\- Excel

\- GitHub



\---



###### \## *Key Metrics*



\- Total Employees: 1470

\- Attrition Count: 237

\- Attrition Rate: 16.12%

\- Average Income: ₹6,503

##### \## Key Insights



\- Overall employee attrition rate is 16.12%.

\- Employees working overtime have a significantly higher attrition rate (30.53%).

\- Employees with 0–2 years of tenure exhibit the highest turnover.

\- Attrition decreases as job satisfaction increases.

\- Employees who left earned approximately 30% less than employees who stayed.

\- Research \& Development and Sales departments contribute the highest number of employee exits.
